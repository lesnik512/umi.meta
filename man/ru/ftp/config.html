Настройки

Хост — имя ftp-хоста.

Логин — логин пользователя для ftp-подключения.

Пароль — пароль пользователя для ftp-подключения.

Порт — порт для подключения (обычно 21).

Директория — путь от домашней директории данного пользователя (оставьте пустым, если хотите использовать домашнюю директорию).

Поля для переноса — список полей через запятую, которые необходимо перемещать на удаленный ftp-сервер.

После заполнения данных при сохранении файлов в поля из списка "Поля для переноса" модуль попытается скопировать данные файлы на ftp-сервер и сохранить текстовую ссылку в поле с названием "{field_name}_ftp". Исходное изображение при этом удалится.

Поля вида "{field_name}_ftp" нужно создать заранее в модуле "Шаблоны данных" или нажав на иконку "Редактировать тип данных" на нужной странице сайта в административной панели.
